# Быстрый старт

Новый проект проще всего создать на основе специализированного репозитория [steroids/boilerplate-nest](https://github.com/steroids/boilerplate-nest/blob/master/README.md).

Следуя инструкциям в README.md, можно несколькими командами получить рабочее NestJS приложение на основе Steroids Nest, с подключенными модулями пользователя, авторизации и работы с файлами.
