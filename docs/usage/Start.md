# Быстрый старт

## Создание нового проекта

Новый проект проще всего создат на основе специализированного репозитория [steroids/boilerplate-nest](https://github.com/steroids/boilerplate-nest/blob/master/README.md).

Следуя инструкциям в README.md, можно насколькими командами получить рабочее NestJS приложение на основе Steroids Nest, с подключенными модулями пользователя, авторизации и работы с файлами.

## Использование в существующих проектах

Для добавления Steroids к существующему проекту необходимо импортировать нужную библиотеку к себе в проект,
а затем использовать необходимый функционал. При этом в случае подключения одного из [модулей Steroids](../modules/Modules.md)
необходимо создать класс модуля с импортированной конфигурацией из главного файла пакета модуля,
выполнить необходимые миграции, настроить файл окружения - в общем, следовать тому, что написано в файле `README.md`
репозитория конкретного модуля.
